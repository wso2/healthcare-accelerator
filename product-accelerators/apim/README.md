
# WSO2 Healthcare API Manager Accelerator

The WSO2 Healthcare API Manager Accelerator provides features tailored for healthcare API management, particularly focusing on FHIR (Fast Healthcare Interoperability Resources), SMART on FHIR, and OAuth 2.0 support.


- [WSO2 Healthcare API Manager Accelerator](distribution/apim-accelerator/README.md)

## WSO2 API Manager Dependencies Reference

This section documents all WSO2 API Manager classes and methods used throughout the Healthcare Accelerator components. This reference helps developers understand the integration points and dependencies on the WSO2 API Manager platform.
